# 날개짓 대작전

웹캠, MediaPipe Pose, HTML Canvas로 만든 60초 체험형 게임입니다. 별도 설치나 빌드 없이 `index.html` 하나로 실행됩니다.

## 실행

카메라는 보안 정책상 `file://`에서 동작하지 않을 수 있으므로 로컬 서버로 실행합니다.

```bash
cd wing-flap-mission
python3 -m http.server 5177
```

브라우저에서 `http://localhost:5177`을 열고 **카메라로 시작**을 누른 뒤 권한을 허용합니다. MediaPipe 파일은 CDN에서 불러오므로 처음 실행할 때 인터넷 연결이 필요합니다.

## 플레이 방법

1. 카메라에서 상반신과 양손이 보이도록 1.5~2m 떨어져 섭니다.
2. 맞춤 설정 화면에서 양팔을 편하게 내리고 약 3초간 기다립니다.
3. 카운트다운 뒤 양손을 어깨 위로 함께 올렸다가 다시 내리며 날개짓합니다.
4. 장애물 통과는 3점, 코인은 10점입니다. 충돌하면 점수가 깎입니다.

`Space` 또는 `↑` 키는 언제든 보조 조작으로 사용할 수 있고, `H` 키를 누르면 도움말로 돌아갑니다. **키보드 데모**는 카메라 없이 부스 안내를 시연할 때 사용합니다.

## 청소년센터 부스 운영 팁

- 노트북 카메라는 참가자의 허리 위와 양손이 모두 들어오도록 정면에 둡니다.
- 역광을 피하고 참가자 얼굴 쪽을 밝게 하면 인식이 안정적입니다.
- 한 번에 한 명만 카메라 앞에 서게 하고, 시작 전 팔을 완전히 내리도록 안내합니다.
- 동작이 잘 잡히지 않으면 도움말로 돌아가 다시 맞춤 설정하거나 키보드 보조 조작을 사용합니다.
- 다음 참가자에게 넘길 때는 **도움말 보기**를 눌러 시작 화면으로 복귀합니다.

## 문제 해결

- 카메라 권한을 거부했다면 브라우저 주소창의 카메라 아이콘에서 권한을 허용한 뒤 다시 시도합니다.
- 카메라 연결 실패 시 `localhost`로 접속했는지, 다른 앱이 카메라를 사용 중인지 확인합니다.
- CDN이 차단된 네트워크에서는 MediaPipe가 로드되지 않으므로 키보드 데모만 사용할 수 있습니다.
